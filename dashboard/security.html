Security Policy

Last updated: 2026-05-02

Authentication and Access

  • API endpoints are protected by API keys (X-API-Key or ?api_key=).
  • Invalid or inactive keys are rejected with authorization errors.
  • RBAC MVP roles are defined as Admin, Analyst, and Read-only.

Encryption and Transport Security

  • TLS is required for hosted traffic.
  • Security headers include HSTS, CSP, X-Frame-Options, and X-Content-Type-Options.
  • Database backups must be encrypted at rest in backup storage.

Logging and Auditability

  • Compliance actions (export, delete, retention changes) are written to the audit log.
  • API key rotation audit event schema is documented for full traceability.
  • Operational traces are available for debugging and post-incident analysis.

Backups and Recovery

  • Backup and restore process is documented with RPO/RTO targets.
  • Restore drill evidence is maintained in enterprise documentation.

Vulnerability Reporting

Security issues can be reported to security@agentlens.one. Process and SLAs are documented in the vulnerability process document.
